# Indian Legal Sections Search

This Rails application allows Indian lawyers to explore various branches of substantive law and find the related legal sections and statutes (Acts) for different legal situations.

## Getting Started

### Prerequisites

- Ruby 3.1+
- Rails 7.2+
- SQLite3

### Installation

1. Clone the repository
2. Run `bundle install`
3. Run `rails db:create db:migrate db:seed`
4. Start the server with `rails server`

### Usage

Visit the root URL to see categories of substantive law. Click on a category (e.g., "Criminal Law") to view all sections in relevant statutes, or search for specific legal issues.
